GoalSetter App

Your personal goal management companion

Features

  • Goal Management: Create, update, and delete your goals easily.
  • Firebase Authentication: Securely sign up and log in users.
  • Push Notifications: Get reminders for important updates or goal deadlines.
  • Custom Dialogs: Beautifully designed popups for goal management.
  • Modern UI: Gradient-rich designs built with Jetpack Compose.

Screenshots

Getting Started

  1. Clone the repository: git clone https://github.com/erpankajkumar53/GoalSetter.git
  2. Open the project in Android Studio.
  3. Configure Firebase:
    • Go to the Firebase Console and download the google-services.json file.
    • Place the file in the app/ directory.
  4. Build and run the app on your device or emulator.

Firebase Integration

Authentication

Supports secure user sign-up, login, and auto-login using Firebase Authentication.

Cloud Messaging

Sends notifications to remind users about important goal updates or deadlines.

Tech Stack

  • Kotlin
  • Jetpack Compose
  • Material3
  • ViewModel for state management
  • Firebase Authentication
  • Firebase Cloud Messaging
